          }

Adobe Escaping

The code below generates HTML in an unsafe way. Prove it by popping an alert.
        
          function escape(s) {
            s = s.replace(/"/g, '\\"');
            return '';
          }

JSON.stringify() Escaping

The code below generates HTML in an unsafe way. Prove it by popping an alert.
        
          function escape(s) {

          }
